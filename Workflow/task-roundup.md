# SOP — 任务汇总 / 催办 / 周清

## 触发条件
每周一次，或老板问"现在都有哪些事在跑"。

## 前置
- `Tasks/open.md` 是任务唯一汇总入口

## 步骤
1. 扫所有公司 `Meetings/` 近期产出 + Inbox 残留，把漏登的任务补进 `Tasks/open.md`。
2. 逐条核对状态：待办 / 进行中 / 阻塞 / 完成。
3. **催办**：截止已过或临近且非「完成」的，列出来，标负责人，提醒老板跟进。
4. **阻塞**：标注阻塞原因，必要时升级到 `Decisions/pending.md`。
5. **周清**：状态=完成的，每月底移到 `Tasks/archive/YYYY-MM-done.md`（不删）。

## 产出
- 一份最新的 `Tasks/open.md` + 一份「需催办/阻塞」清单给老板。

## 检查清单
- [ ] 每条任务有归属公司、负责人、截止
- [ ] 逾期项已挑出来催
- [ ] 完成项按月归档，open 只剩活跃
- [ ] 跨公司只放标题，敏感细节留各公司（防串味）
