# SOP — 月度财务对账

## 触发条件
每月初，给上个月各公司财务做快照并汇总。

## 前置
- 各公司当月报表/对账数据齐了
- 货币默认 AUD，跨币种标注（铁律）

## 步骤
1. 逐家公司：在 `Companies/<x>/Finance/snapshot.md`
   - 把旧快照内容移到「历史快照」一行（不覆盖）
   - 写新快照：现金 / 月收入 / 月支出 / 月净 / 跑道 / 主要应收应付
   - 追加变更记录 `YYYY-MM-DD · 变了什么`
2. 回 `Cashflow/latest.md`：把旧 `latest` 内容存为 `Cashflow/archive/YYYY-MM.md`，再写本月最新汇总表（各公司一行 + 合计）。
3. 异常（跑道<3 个月、现金骤降）→ 写进 `Decisions/pending.md` 提醒老板。

## 产出
- 各公司当月财务快照 + Cashflow 汇总更新 + 历史归档。

## 检查清单
- [ ] 旧快照保留为历史，没被覆盖（铁律 #3）
- [ ] 各公司财务**没有互相串入**
- [ ] 货币标注清楚
- [ ] 跑道告警已上报
