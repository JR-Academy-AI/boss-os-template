# SOP — 收件箱分拣

## 触发条件
`Inbox/` 里堆了原始件（邮件、截图、转发文件、随手记），要归位。

## 前置
- 逐件判断归属公司；**不确定就问老板，绝不猜**（铁律 #1 防串味）

## 步骤
逐件处理，按内容分流：
1. **会议纪要** → 走 `meeting-notes.md`，进对应公司 `Meetings/`。
2. **财务**（发票/报表/对账）→ 对应公司 `Finance/`，更新快照；涉及 GST/ABN 按澳洲税务规则。
3. **任务/待办** → `Tasks/open.md`，标归属公司+负责人+截止。
4. **决策请求** → `Decisions/pending.md`。
5. **未来安排** → 工作类进 `Personal/Agenda/`，生活类进 `Personal/Events/`。
6. **纯参考资料** → 对应公司 overview 或单独存档。
7. 处理完的原始件移到 `Inbox/_done/`，**不删**（铁律 #2）。

## 产出
- Inbox 清空到 `_done/`，内容各归各位，汇总区已更新。

## 检查清单
- [ ] 每件都判明归属公司（没猜）
- [ ] 原始件保留在 _done/
- [ ] 没有跨公司串味
