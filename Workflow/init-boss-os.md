# SOP — 给一个老板快速建 OS（盖章）

## 触发条件
要给某个老板从零做一套多公司 OS。

## 步骤
1. 整包复制 `boss-os-template/` → 改名 `{boss}-os/`。
2. 问老板 5 个问题，填 `PROFILE.md`（用 PROFILE.template.md）：
   - 你叫什么、什么角色？
   - 管几家公司，分别叫什么、你在里面是什么角色？
   - 每家要不要管财务？
   - 你的决策风格 / 沟通偏好？
   - 每周最想一眼看到什么？
3. 对每家公司：复制 `Companies/_template/` → 改名为公司代号，填 `overview.md`。
4. 删掉用不到的：不管财务就删该公司 `Finance/`。
5. 告诉老板：以后东西丢进 `Inbox/`，开会纪要丢进对应公司 `Meetings/`，AI 负责整理 + 汇总。

## 产出
- 一套可直接用的 `{boss}-os/`，老板当天就能往里丢东西。

## 检查清单
- [ ] PROFILE 填完，公司表对得上目录
- [ ] 每家公司目录建好
- [ ] 铁律 #1（不串味）老板知情
