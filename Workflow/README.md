# Workflow — 老板 OS 通用 SOP 索引

做重复性的事前，先来这里找有没有现成 SOP。每个 SOP follow：触发条件 / 前置 / 步骤 / 产出 / 检查清单。

| SOP | 什么时候用 |
|-----|-----------|
| `init-boss-os.md` | 给一个新老板从零盖章建 OS |
| `add-company.md` | 新增/接手一家公司 |
| `inbox-triage.md` | Inbox 堆了原始件要归位 |
| `meeting-notes.md` | 开完会，整理会议记录 → 决策+任务 |
| `task-roundup.md` | 每周任务汇总/催办/周清 |
| `weekly-review.md` | 每周一生成全局周报 |
| `monthly-finance.md` | 每月各公司财务对账 + 汇总 |
| `archive-rotation.md` | 定期把 live 文件滚动归档，防撑爆 |
