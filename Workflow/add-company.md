# SOP — 开一家新公司

## 触发条件
老板新增/接手一家公司，要纳入 OS 管理。

## 前置
- 想好公司代号（短、唯一，如 `company-c`）

## 步骤
1. 复制 `Companies/_template/` → 改名为代号。
2. 填 `overview.md`：定位 / 现状 / 我的角色 / 关键人 / 当前季度目标。
3. 要管财务就保留 `Finance/`、填首个快照；不管就删掉 `Finance/`。
4. 回 `PROFILE.md` 的「我管的公司」表加一行。
5. 回 `Cashflow/latest.md` 加一行（若管财务）。
6. 告诉老板：这家的会议纪要往 `Companies/<代号>/Meetings/` 丢。

## 产出
- 一家可用的新公司目录 + PROFILE/Cashflow 同步登记。

## 检查清单
- [ ] 代号唯一，PROFILE 表和目录名一致
- [ ] overview 填完
- [ ] 不管财务的已删 Finance/
- [ ] 老板知道这家的隔离边界（铁律 #1）
