# Inbox — 杂项原始件中转站

**只放需要归档的零散原始件**：邮件、报表截图、转发的文件、随手记、名片。
这是个临时中转站，归完档就清走。

> ⚠️ 不是会议、不是日程。
> - 已开完的会 → 直接进对应公司 `Companies/<x>/Meetings/`，不经过这里。
> - 未来要发生的事 → `Personal/Agenda/upcoming.md`。

## AI 的职责（老板只管丢，不管分类）
1. 判断属于哪家公司（不确定就问，**绝不猜** —— 防串味）。
2. 财务类 → 进对应公司 `Finance/`，更新快照。
3. 里面提到的任务/决策 → 汇总到 `Tasks/open.md` / `Decisions/pending.md`。
4. 提到未来安排 → 进 `Personal/Agenda/upcoming.md`。
5. 原始文件**保留**（铁律 #2）：处理完移到 `Inbox/_done/`，不删。

命名：`YYYY-MM-DD-简短描述.md`
